# Suyog+ Job Finder

A **Streamlit app** to help persons with disabilities in India find suitable jobs.

## Features
- Filter jobs by:
  - Type of disability
  - Subcategory (Intellectual & Developmental Disabilities)
  - Qualification
  - Department
  - Functional activities
- Display matched jobs in a table
- Download job list as PDF
